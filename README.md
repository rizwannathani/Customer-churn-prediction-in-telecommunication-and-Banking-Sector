# 📉 Customer Churn Prediction using Machine Learning & Deep Learning

## 📌 Business Problem

Customer churn is a major challenge for industries such as telecommunications and banking, where losing existing customers can significantly impact revenue. Businesses need reliable predictive models to identify customers at risk of leaving, enabling proactive retention strategies and reducing churn.

---

## 🚀 Project Overview

This project develops an end-to-end customer churn prediction using **Machine Learning** and **Deep Learning** techniques on telecom and banking datasets. It covers the complete data science workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, feature selection, handling class imbalance with SMOTE, model training, and performance evaluation. Multiple models were compared to identify the most effective approach for predicting customer churn across different industries.
